## Объявление
```php
<?php 
$arr = [1 => 'пн', 2 => 'вт', 3 => 'ср', 4 => 'чт', 5 => 'пт', 6 => 'сб', 7 => 'вс']; 
?>
```

```php
<?php 
echo $arr[1]; // выведет 'пн' 
?>
```

## Хитрость с ключами в PHP
```php
<?php 
$arr = [1 => 'пн', 'вт', 'ср', 'чт', 'пт', 'сб', 'вс']; 
echo $arr[1]; // выведет 'пн' 
echo $arr[2]; // выведет 'вт' 
echo $arr[3]; // выведет 'ср' 
?>
```
