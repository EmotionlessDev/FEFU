## empty
Зачастую в скриптах возникает нужда в проверке переменной на _пустоту_. В PHP переменная будет пустая, если она равна `0`, `''`, `'0'`, `false` или `null`.

Проверка на пустоту выполняется с помощью команды `empty`:

```php
<?php 
$test = ''; 
if (empty($test)) { 
echo '+++'; 
} else { 
echo '---'; 
} 
?>
```
